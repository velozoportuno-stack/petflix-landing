Política de Privacidade

Última atualização: 30 de março de 2026

A Mundo PetFlix compromete-se a proteger a tua privacidade. Esta Política de Privacidade descreve como recolhemos, utilizamos e protegemos os teus dados pessoais quando visitas o nosso website ou adquires os nossos produtos.

1. Responsável pelo Tratamento de Dados

Mundo PetFlix
Email de contacto: info@mundopetflix.pt

2. Dados que Recolhemos

Recolhemos apenas os dados estritamente necessários para processar a tua encomenda e entregar o produto digital:

  • Nome e endereço de email (para entrega do produto)
  • Dados de pagamento (processados de forma segura pela Stripe — não armazenamos dados de cartão)
  • Dados de navegação anónimos (para análise de desempenho do website)

3. Como Utilizamos os Teus Dados

Os teus dados pessoais são utilizados exclusivamente para:

  • Processar e confirmar o teu pagamento
  • Entregar o produto digital por email
  • Responder a pedidos de suporte ou reembolso
  • Enviar comunicações relacionadas com o produto, caso autorizes

4. Partilha de Dados com Terceiros

Não vendemos nem partilhamos os teus dados pessoais com terceiros para fins comerciais. Os dados de pagamento são processados pela Stripe, Inc., em conformidade com os mais elevados padrões de segurança (PCI DSS). Podes consultar a política de privacidade da Stripe em stripe.com/privacy.

5. Retenção de Dados

Conservamos os teus dados pelo período mínimo necessário para cumprir as obrigações legais e contratuais — geralmente 5 anos para registos de faturação, em conformidade com a legislação portuguesa.

6. Os Teus Direitos (RGPD)

Ao abrigo do Regulamento Geral sobre a Proteção de Dados (RGPD), tens o direito de:

  • Acesso: solicitar uma cópia dos teus dados pessoais
  • Retificação: corrigir dados incorretos ou incompletos
  • Apagamento: solicitar a eliminação dos teus dados
  • Portabilidade: receber os teus dados num formato legível por máquina
  • Oposição: opor-te ao tratamento dos teus dados para fins de marketing

Para exercer qualquer um destes direitos, contacta-nos em info@mundopetflix.pt.

7. Cookies

Este website pode utilizar cookies técnicos essenciais para o funcionamento correto da página. Não utilizamos cookies de rastreamento de terceiros sem o teu consentimento.

8. Segurança

Adotamos medidas técnicas e organizacionais adequadas para proteger os teus dados contra acesso não autorizado, perda ou divulgação indevida.

9. Alterações a esta Política

Podemos atualizar esta Política de Privacidade periodicamente. Sempre que o fizermos, atualizamos a data no topo desta página. Recomendamos que a consultes regularmente.

10. Contacto

Se tiveres alguma dúvida sobre esta Política de Privacidade, entra em contacto connosco:
📧 info@mundopetflix.pt
